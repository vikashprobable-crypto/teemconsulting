# Uploads Directory

This directory contains uploaded images for:
- Client logos (`/uploads/logos/`)
- Gallery images (`/uploads/gallery/`)
- Team member photos (`/uploads/team/`)
- Other assets

## Directory Structure:
```
uploads/
├── logos/
├── gallery/
├── team/
└── assets/
```

## Notes:
- Images are automatically organized by type
- File names include timestamps for uniqueness
- Maximum file size: 10MB
- Supported formats: JPG, PNG, GIF, WebP
